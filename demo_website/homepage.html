Welcome to SQLancer

SQLancer is an open-source tool that can automatically generate and execute test cases for SQL database management systems (DBMSs). It can help detect bugs and vulnerabilities in DBMSs, making it an important tool for software testing.

Supported DBMS

  • MySQL
  • PostgreSQL
  • Oracle
  • SQLite
